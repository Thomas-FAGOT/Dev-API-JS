L'objectif de ce projet est de créer une API via Java Script

Grace à la bare de navigation vous pouvez utiliser 3 requêtes différentes

Le Get One : Il permet de récuper les inofrmations d'un article via son ID

Le Get All : Il permet de récuper les inofrmations de tous les articles

Le Post One : Il permet de poster un nouveau article en lui donnant un nom et un prix
